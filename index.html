Projects

Internet Relay Chat Protocol

This RFC outlines an IRC client server application. This system facilitates the group of users that are subscribed to the room receive messages of the other users in the room. Any message sent to that room is forwarded to all users currently joined to that room. Users can also send private messages directly to other users.

Tuberculosis Detection From Chest X-RAY

In this project we have used Local KNN, Weighted KNN and Gradient Boosting to predict the disease in patients and we have compared their accuracies and found the best for our problem.
